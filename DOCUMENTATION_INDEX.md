1. اقرأ README_AR.md (10 دقائق)

3. عدل بعض محتويات البيانات (5 دقائق)
4. اقرأ DEVELOPER_GUIDE.md (30 دقيقة)

1. اقرأ PROJECT_STRUCTURE.md (20 دقيقة)
2. استكشف البنية الفعلية (10 دقائق)
3. اقرأ DEVELOPER_GUIDE.md (30 دقيقة)
4. جرب إنشاء مكون جديد (30 دقيقة)
